## 按键与宏,提高操作 ##
### 基本宏 ###
```
#showtooltip 枯萎凋零
/cast !枯萎凋零
```
### 主动取消增益 ###
```
/cancelaura 风火雷电
/cancelaura 剑刃风暴
/cancelaura 圣盾术
/cancelaura 保护之手
/cancelaura 威慑
/cancelaura 进食
/cancelaura 喝水
/dismount [mounted]
```
### 冲锋断筋宏 ###
```
#showtooltip 冲锋
/cast 断筋
/cast [help]援护;冲锋
```
### 姿态切换宏 ###
```
#showtooltip 战斗姿态
/use [stance:1]防御姿态;[stance:2/3]战斗姿态
```
### 远距离视角 ###
```
/script SetCVar("cameradistanceMax",50)
/script SetCVar("cameradistanceMax",100)
```
### 技能加上召唤坐骑 ###
```
#showtooltip 寒冰锁链
/cast [flyable,nocombat]阿彻鲁斯死亡战马;[outdoors,nocombat]阿彻鲁斯死亡战马;[harm,combat]寒冰锁链
```
### 鼠标指向和进攻技能目标判断优先级宏(奶妈常用,我自己实在是用不习惯) ###
```
#showtooltip 技能
/cast [@mouseover,help,exists]技能;技能
#showtooltip 神圣之火
/cast [@focus,harm,exists]神圣之火;[@targettarget,harm,exists]神圣之火; [@target,harm,exists]神圣之火
```
### 小D爆发宏 ###
```
#showtooltip 超凡之盟
/cast 超凡之盟
/cast 自然的守护
/cast 月火术
/cast 阳炎术
```

```
/cast [button:1] 正义之怒

```
## 远离Addons,培养意识 ##
插件这个东西如果是热爱PVP活动的,尽量只使用wow提供的界面元素自定义的功能宏,{% label 多练习少依赖插件 danger %}.
如果是PVE向的,却需要自己多花些时间学习(毕竟脑子不够用啊),调得花哨或者清爽都没关系,自已玩得舒服最重要.
### PVP线下赛也允许使用,自己动手写个谈不上插件的插件 ###
#### XyongQ.toc ####
``` 
## Interface: 60000
## Title: XYQ
## Author: 徐骞
## Notes: 常用的WOW原生界面调整函数.|n|cFF00FF00Author: |r徐骞
## Version: v1.1
XyongQ.lua
```
#### XyongQ.lua ####
```
if select(2, UnitClass("player")) ~= "WARRIOR" then return end
TargetFrameSpellBar:SetScript("OnShow", nil) 
TargetFrameSpellBar:SetParent(UIParent) 
TargetFrameSpellBar:ClearAllPoints() 
TargetFrameSpellBar:SetPoint("CENTER",UIParent,"CENTER",8,132)
TargetFrameSpellBar:SetScale(2.0)
```
这只是一个改变{% label 目标施法条 danger %}的例子,教你如何把在游戏中用宏改变界面元素写在插件里,不用每次进游戏里面点击宏.
更多lua插件编写教程,请[点击1](http://bbs.ngacn.cc/read.php?tid=3573145&_fp=2)以及[点击2](http://bbs.ngacn.cc/read.php?tid=4671136).
